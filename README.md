# Thinking, Fast and Slow — A Field Guide

An interactive companion to Daniel Kahneman's *Thinking, Fast and Slow*. A single, dependency-free `index.html`:

- **Concepts** — 28 core terms with definitions, "why it matters," and chapter references
- **Concept map** — a force-directed graph of how the ideas connect (click any node)
- **Bias library** — 25 biases in 4 families, each with mechanism, real-world examples, the original experiments, and a "how to catch it" tip
- **Spot-the-bias quiz** — 13 story-driven scenarios with near-miss distractors
- **Math corner** — the five load-bearing statistical ideas + an interactive prospect-theory value curve
- **Roadmap** — a staged path deeper into behavioral economics

## Run locally
Just open `index.html` in a browser. No build step, no dependencies (fonts load from Google Fonts).
